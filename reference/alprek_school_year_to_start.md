# Convert School Year to Start Year

Extracts the first year from a "YYYY-YYYY" school year string.

## Usage

``` r
alprek_school_year_to_start(school_year)
```

## Arguments

- school_year:

  Character. School year in "YYYY-YYYY" format.

## Value

Integer start year.

## Examples

``` r
alprek_school_year_to_start("2024-2025")
#> [1] 2024
# 2024L
```
